# UI/UX Specs (Morphism)

- **Primary Color:** `#084619` (Dark Green) -> Use for highlights and buttons.
- **Secondary Color:** `#000000` (Rich Black) -> For text/headings.
- **Background:** `#FFFFFF` (White).
- **Surface:** `#F9FAFB` (Off-white for input fields).
- **Typography:** Inter or System Sans-serif (Clean, Modern).
-
- **Buttons:** Rounded-lg (8px), bold text.
