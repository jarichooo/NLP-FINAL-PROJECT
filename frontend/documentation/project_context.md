# Project: Morphism

## Purpose

A high-end Tagalog-English (Taglish) Grammar Detection and Correction web application. It uses a hybrid approach: RoBERTa for error detection and a rule-based engine for morpho-syntactic correction.

## Core Features

- Split-screen Editor: Left (Input), Right (Corrected Output).
- Real-time Detection: Underlining anomalies as the user types.
- Transparency: Users can inspect why a rule was applied (Rule-based engine).
- We will only focus on FRONTEND and make it ready for backend connection
